Experience Lab Objectives & Premise

Objectives

Note

The lab focuses on only the execution stage of the testing lifecycle and not in its entirety.

  • Demonstrate a successful continuous testing implementation with Azure native services integrating with open source testing tools
  • Identify types and stages of testing are essential for end-to-end quality assurance
  • Choose various tooling alternatives, enablers and best practices in testing for DevOps
  • Build and execute the continuous testing pipeline covering infra and pipeline as a code
  • Leverage the computing power of containers for seamless scalability in automation
  • Debug typical issues and fix your continuous testing pipeline

Lab Structure

Scope

This lab constitutes five stages namely, Pre-Build QA, Post-Build QA, Functional Assurance, Experience Assurance, End-to-End Execution that explain a continuous testing pipeline in a chronological sequence.

Modular Structure

Each stage provides an overview introducing concepts and benefits, followed by steps to replicate, before getting into the pipeline execution.

Solve & Learn

In each stage, we have also induced few defects in the application code and test assets, so that you can debug and fix them to gain a more practical insight into this workshop.

Thought Leadership

Additionally, we have provided relevant thought leadership content for reference with best practices and tooling alternatives to consider in each pipeline stage.

In order to get the best out this workshop, we recommend you follow the sequence and build your pipeline in progression.

Learnings

On completion of all lab stages, you will be able to -

  • Differentiate continuous testing from traditional testing
  • Propose key testing types for ensuring end-to-end quality assurance
  • Build and execute the continuous testing pipeline
  • Integrate open source automation tools in the pipeline
  • Help debug typical issues and fix the continuous testing pipeline
  • Leverage tooling alternatives, enablers and best practices in continuous testing
