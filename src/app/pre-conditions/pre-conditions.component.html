How to create an Azure Account?

Note
  • To create account on Azure Portal, you need to have a phone number, a Credit / Debit card, and a GitHub account (or) a Microsoft account.
  • If you already have an Azure account - Free (or) Pay as you go Subscription and have Administrator access, you can skip this page. Also please go through the Azure Free Account FAQ section for more information on your account, refer https://azure.microsoft.com/en-us/free/free-account-faq/

2. Click on Start Free as indicated in the Exhibit below:

3. This will result in you navigating to the Sign-In Page. Enter existing Personal Email ID and click on Next as per the Exhibit below:

4. Enter Password and click on Sign-In Button, refer the Exhibit below:

5. “Create your Azure free account” Page will be displayed as shown in the Exhibit below to capture sign-up information. Fill up the details in your profile section and click on Next Button.

Enable Test Plan:

1. On https://dev.azure.com/yourorganization page, Navigate to Organization Settings

2. Organization Setting Page will be displayed, now select Billing & select Start Free Trial

3. Select Start free trial on the pop-up

4. Go to Users section and change access level to Basic + Test Plan

5. Select Basic + Test Plans from dropdown
